## 의미를 알 수 없는 이름

```java
// 코드 1.1
class MemoryStateManager {
    void changeIntValue01(int changeValue) {
        intValue01 -= changeValue;
        if (intValue01 < 0) {
            intValue01 = 0;
            updateState02Flag();
        })
    }
    ...
}
```

코드 1.1을 보면 이 로직을 무엇을 의미하는지 알 수 있나요?
